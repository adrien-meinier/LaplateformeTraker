# La Plateforme Tracker 

### Overview
La Plateforme Tracker is a simple monitoring app for a list of students and their grades. 
Build to learn and illustrate : 
- Java and JavaFX, with a Maven build. 
- A Model View Controller architecture.
- OOD in Java.
- Unitary tests and code coverage with Jutil and Jacoco rapports.
